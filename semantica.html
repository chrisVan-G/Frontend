Encabezado

El encabezado de una sección de contenido o la página web. El encabezado de la página web suele contener la marca o el logotipo del sitio web.

Navegación

Los enlaces de navegación de una sección o la página web

Semántica HTML

Main

Especifica el contenido principal de una sección o de la página web.

Aparte

Conjunto de contenidos secundarios que no son necesarios para comprender el contenido principal.

Artículo

Bloque de contenido independiente y autónomo, como una entrada de blog o un producto.

Sección

Sección independiente del documento, que suele utilizarse dentro de los elementos body y article.

Detalles
  • Es una sección de contenido contraída
  • que puede expandirse
  • si el usuario desea verla.
Resumen

Especifica el resumen o la leyenda de un elemento details.
Pie de página

El pie de página de una sección de contenido o la página web. En una página web, suele contener enlaces secundarios, el aviso de copyright, la política de privacidad y enlaces a la política de cookies.
